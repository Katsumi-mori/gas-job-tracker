# 選考管理ツール（GAS）

就活の選考状況をスプシで管理し、毎朝Gmailで通知するGASツール。

## 機能
- 登録中の企業一覧をメール通知
- 提出期限3日前にリマインド
- 提出期限3日前にSlack通知
- 時間トリガーで毎朝自動実行
- Googleフォーム送信時にSlack通知（onFormSubmitトリガー）

## ファイル
- `sendTestMail.gs` - 選考管理（メール・Slack通知）
- `onFormSubmit.gs` - フォーム送信時の自動Slack通知 ← New!

## 使い方
1. Googleスプレッドシートを作成
2. 拡張機能 → Apps Script でスクリプトエディタを開く
3. このファイルの内容をコピペ
4. トリガーで自動実行を設定

## セットアップ
1. Googleスプレッドシートを作成
2. 拡張機能 → Apps Scriptでコードを貼り付け
3. プロパティサービスに `SLACK_WEBHOOK_URL` を登録
4. トリガーで自動実行を設定

## 使い方の補足
初めて使う方は、以下の手順でスプレッドシートを準備してください。

1. Googleドライブから新しいスプレッドシートを作成し、わかりやすい名前（例: 「選考管理」）を付ける
2. 1行目に見出し行として「会社名」「フェーズ」「提出期限」「メモ」などの列を用意する
3. 2行目以降に登録したい企業の選考情報を入力する（日付はYYYY/MM/DD形式を推奨）
4. メニューの「拡張機能」→「Apps Script」を開き、本リポジトリの`.gs`ファイルの内容を貼り付ける
5. スクリプトのプロパティに`SLACK_WEBHOOK_URL`を設定し、必要に応じてトリガー（時間主導型・フォーム送信時）を登録する
